#### 关于：
这是一个带有雨后泥土湿润气息的着色器，具有像素效果和噪点。  
这是我编写的第五个着色器包，它基于我的[Lantern](https://modrinth.com/shader/lantern-shader)着色器修改而来，希望你能喜欢(^-^)  

#### 性能：
并不需要多高配置的电脑，核显就足以流畅运行  
视距设置推荐8个区块体验最好，可以更少，但不宜过多，一般8～11个就好。
